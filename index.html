Instructions

Quick start:

  1. Select which sites you want to edit.
  2. Enter the root path to the relevant folders.
  3. Copy the generated code to the terminal.

Setup:

  1. Enter the name of the Alpine folder where the repo is located. This assumes a root directory structure of /usr/local/alpine/ [default: alpine-git]
  2. Choose a file type [default: html]
  3. Select whether you have subl installed [default: yes]

Once you run the tool, your preferences will be saved to a cookie.
